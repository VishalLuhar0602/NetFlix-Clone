Welcome to Project SportSync360

Project SportSync 360 (P-SS360) is an initiative to centralized sports events. It would help in reshaping the management of sports.

Features It allows organizers to create new events, delete old events and edit current events in real time. This also provides platform coaches that allows them to monitor and communicate with their athletes, further aims to help players by giving them valuable feedback about their platform.
